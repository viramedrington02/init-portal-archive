# init-portal-archive

## 项目简介

init-portal-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为 HTML 页面资源的集中存储与展示空间。

## 目录结构

```
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的静态资源（如 CSS、JS、图片等）
├── index.html      # 仓库首页或导航入口
└── README.md       # 本文件
```

## 页面归档说明

- 所有归档页面均放置于 `pages/` 目录下，按需组织子目录。
- 每个页面应为完整的独立 HTML 文件，包含必要的样式与脚本引用。
- 资源文件建议统一存放于 `assets/` 目录，便于管理与引用。
- 页面内容应保持原始结构与功能，不进行额外修改或注入。

## 使用方式

1. 克隆本仓库到本地。
2. 将需要归档的 HTML 页面及相关资源放入对应目录。
3. 可通过本地服务器或直接打开 `index.html` 浏览归档内容。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面，或修复现有页面问题。
- 提交前请确保页面功能正常，资源路径正确。
- 本仓库不承担页面内容的审核义务，但建议提交内容保持合法合规。
- 如需删除或更新页面，请提交 Issue 或直接发起 PR。

## 许可

本仓库仅作为页面归档与展示使用，所有页面内容版权归原作者所有。  
仓库本身无特定许可证，请尊重原始内容的版权与使用条款。
